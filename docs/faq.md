## Frequently Answered Questions

### What's MoonlightBot's prefix?

Look at the bot's "playing" status. Generally, the prefix is `m:` for the standard bot, `n:` for beta and `p:` for preview.

Mentioning the bot will work in any case, like `@MoonlightBot#9699 help` will work regardless of prefix.

### Why is MoonlightBot not responding to my commands?

That might be because of the following situations:

  - You're trying to use a command that can only be used in a server (like configuration and moderation ones) in a Direct Message
  - The server owner disabled permission errors, and either:
    - The command's level is higher than your level
    - The command is disabled
    - Commands have been disabled in the channel you're trying to run it in. Perhaps that server has a specific channel for using bots?

  - If the problem persists after trying all of those solution, contact support.

# MoonlightBot

MoonlightBot is a Discord bot that offers tools to help server moderators with tasks like storing infractions, removing a
ban after a set time, delete messages in mass, etc... it can be a good replacement for situations where Discord's default
permissions are too broad to be used, or you want to be safe from your moderators' accounts being exploited.


## Getting started

You can add MoonlightBot in any Discord server and start configuring it immediately.

?> If the server of your choice does not appear there, make sure you have “Manage Server" permission in that server


## Configuration

MoonlightBot has an extremely flexible configuration that allows you to disable or restrict any feature at any time, such as
disabling commands from being used in specific channels, or disabling a speciﬁc command altogether.
Channels can also be set to log actions in them in a granular manner.

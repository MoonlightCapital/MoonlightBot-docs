# Ban command reference
The `ban` command is used to ban an user from the server it's used in. It works even if the user is not in the server.

## Arguments
| Name   | Type | Description                                                                                                                                    |
|--------|------|------------------------------------------------------------------------------------------------------------------------------------------------|
| user   | User | The user you want to ban from the server                                                                                                       |
| reason | Text | *Optional.* The reason for the ban that will be supplied in the logs. Can contain spaces and will be truncated to 1024 characters if too long. |
